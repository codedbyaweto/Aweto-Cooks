# 🍴 Aweto Cooks

A modern, responsive React application to manage your recipes.  
Add, search, filter, and view recipes in a clean, professional interface with dark mode vibes, gradient UI, and intuitive navigation.

## Features

- Add, edit, and remove recipes
- Search recipes by name
- Filter recipes by category (Breakfast, Lunch, Dinner, Snack)
- Recipe detail view with ingredients
- Modern UI with dark container, gradient background, and red action buttons
- Responsive layout for mobile and desktop
- Login / Logout system
- Smooth navigation with React Router

## Tech Stack

- **Frontend:** React, TypeScript, React Router
- **State Management:** useState, useMemo
- **Styling:** CSS3 (modern dark theme with gradients)
- **Performance:** Lazy-loaded pages (React.lazy + Suspense)
